# ⚔️ PlanYourDay — Craft Your Quest

> A Minecraft-themed daily planner, diary, to-do list & reminder app — gamified like an RPG adventure.

![PlanYourDay Banner](https://img.shields.io/badge/PlanYourDay-Craft%20Your%20Quest-6DAF34?style=for-the-badge&logo=data:image/png;base64,)

---

## 🎮 Features

### 📜 Adventurer's Log (Daily Diary)
- Write daily journal entries with mood selectors (😊 🔥 😴 😤 🏆 🤔)
- Entries saved and displayed in a pixel-art styled scroll log
- Earn **+15 XP** per diary entry

### ⚔️ Quest Board (To-Do List)
- Three quest tiers: **Today**, **Weekly**, and **Epic**
- Priority levels: 🔴 High (+30 XP), 🟡 Medium (+20 XP), 🟢 Low (+10 XP)
- Complete quests to earn XP, trigger confetti bursts, and level up!
- Completed quests move to the **Done** archive

### 🕯 Torches (Reminders)
- Set named reminders with specific times
- Reminders fire toast notifications at the set time
- Live torch flame animation for active reminders

### 🏆 RPG Gamification
| Feature | Description |
|---|---|
| XP Bar | Fills as you complete quests and write entries |
| Level System | Level up with increasing XP thresholds |
| Achievement Badges | 8 unlockable badges (First Quest, Diary Keeper, Level 5, On Fire, etc.) |
| Health Hearts ❤️ | Reflects total XP earned |
| Hunger Hearts 🍗 | Reflects diary entries written |
| Inventory Hotbar | Live counts of quests done, reminders, entries, and pending tasks |
| Day Streak 🔥 | Tracks consecutive days of diary writing |

## 🚀 Getting Started

### Option 1 — Open Directly
Just open `index.html` in any modern browser. No server needed.

### Option 2 — GitHub Pages
1. Fork or upload this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Visit `https://yourusername.github.io/planyourday`

- **Pixel art aesthetic** inspired by Minecraft's UI

- Minecraft block colors: grass green, dirt brown, diamond blue, gold, redstone red
